Prerequisites

  • If you use my plot program to produce graphs, you will need the Perl GD and Chart modules Chart-0.99c-pre3.tar.gz from CPAN.
  • If you use the supplied minibench script for measuring CPU and Memory usage, you will need the Perl GTop module from CPAN.

To download a file from this directory, you should be able to right click on its link and choose "Save As"

(Note: The shootout is compressed with the bzip2 utility to save space and download time.)
