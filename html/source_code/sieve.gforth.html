    6 \ find and count all primes from 2 to 8192
    7 
    8 decimal
    9 
   10 \ read NUM from last command line argument
   11 0. argc @ 1- arg >number 2drop drop constant NUM
   12 
   13 \ we search for primes up to this SIZE
   14 8192 constant SIZE
   15 
   16 \ Flags is an array of chars of length SIZE
   17 \ we'll mark all non-prime indexes in this array as false
   18 \ the remaining indexes will be prime numbers
   19 create Flags SIZE allot
   20 
   21 \ EndFlags points to end of array Flags
   22 Flags SIZE + constant EndFlags
   23 
   24 \ FLAGMULTS
   25 \ flag all multiples of n in array as not prime
   26 \ array has address range: fromaddr toaddr
   27 \ starting value for fromaddr should be
   28 \   arraystart n n + +
   29 : flagmults ( n toaddr fromaddr -- )
   30     do
   31     0 i c! dup
   32     +loop ;
   33 \ END FLAGMULTS
   34 
   35 
   36 \ PRIMES
   37 \ find all primes from 2 to SIZE
   38 : primes  ( -- n )
   39 \ fill array Flags with 1's
   40     Flags SIZE 1 fill
   41     0 2
   42     \ index i ranges from Flags to EndFlags
   43     EndFlags Flags
   44     do
   45     i c@
   46     \ If the current Flags[i] is true (i.e. i is prime)
   47     if
   48         dup i + dup EndFlags <
   49         \ If we aren't at end of flags array yet
   50         if
   51         EndFlags swap flagmults
   52         else
   53         drop
   54             then
   55         \ Increment our Count of Primes
   56             swap 1+ swap
   57     then
   58     1+
   59     loop
   60     drop \ your pants!
   61     ;
   62 \ END PRIMES (Returns: Count)
   63 
   64 \ BENCHMARK
   65 \ run the test NUM times
   66 : benchmark  0 NUM 0 do  primes nip loop ;
   67 
   68 
   69 \ now print count of how many Flags are now "true"
   70 ." Count: " benchmark  1 u.r cr
   71 
   72 
   73 \ PPRIMES
   74 \ for testing, we can print out all the prime numbers
   75 : pprimes ( -- )
   76     SIZE 0 do Flags i + c@ if i 2 + . then loop cr ;
   77 
   78 \ uncomment the following to print the primes or debug
   79 \ pprimes
   80 \ flags 100 dump
