    9 <h4>About this test</h4>
   10 <p>For this test, each program should be implemented in the <a
   11   href="../../method.php#sameway"><i>same way</i></a>.</p>
   12 <p>This is a highly contrived test that exercises various <i>list</i>
   13   operations: pushing and popping from either end of a list, list
   14   reversal, list copying, and comparing lists for equality.  We are
   15   not testing list traversal unless it is implicitly implemented in
   16   those places where we allow builtin list functions.</p>
   17 <p>The test can use whichever sequence data structure is best suited
   18   for the problem (e.g. list, array, deque, or doubly-linked-list).
   19   It is preferred to use a native data structure, if possible, but a
   20   number of languages benefit from creating a custom data structure.</p>
   21 <p>Here's a description of how each test program works:</p>
   22 <ul>
   23   <li>first create a list (L1) of integers from 1 through SIZE (SIZE is
   24   currently defined as 10000).</li>
   25   <li>copy L1 to L2 (can use any builtin list copy function, if available)<
   26                      /li>
   27   <li>remove each individual item from left side (head) of L2 and append
   28                                                   to
   29   right side (tail) of L3 (preserving order).  (L2 should be emptied by
   30   one item at a time as that item is appended to L3).</li>
   31   <li>remove each individual item from right side (tail) of L3 and append
   32   to right side (tail) of L2 (reversing list).  (L3 should be emptied
   33   by one item at a time as that item is appended to L2).</li>
   34   <li>reverse L1 (preferably in place) (can use any builtin function for
   35   this, if available).</li>
   36   <li>check that first item of L1 is now == SIZE.</li>
   37   <li>and compare L1 and L2 for equality and return length of L1 (which
   38   should be equal to SIZE).</li>
   39 </ul>
